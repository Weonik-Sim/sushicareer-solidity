#### Contract 設定

```
yarn install
```

#### Contract Compile

```
npx hardhat compile
```

#### Contract 起動（localhost)
```
npx hardhat node --network hardhat
```

#### Contract Deploy
```
npx hardhat run scripts/deploy.js --network localhost
```

#### Contract Deploy - Scroll Testnet
```
npx hardhat run --network scrollTestnet scripts/deploy.js
```
